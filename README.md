# To-do App

React Native to-do application.

##  Features
- Add/delete/edit tasks
- Dark/light theme
- Local data storage
- Modern design

## Usage
- **Add Task:** Click "Add Task" button
- **Edit:** Click on task
- **Delete:** Click trash icon
- **Theme:** Click sun/moon icon on top right

## Technologies
- React Native
- Expo
- AsyncStorage
